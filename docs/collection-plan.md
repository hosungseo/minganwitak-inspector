# 수집 계획

## 목표
중앙행정기관 홈페이지에 흩어진 민간위탁 공개 정보를 수집해 표준화한다.

## 1차 시범 기관
- 보건복지부
- 행정안전부
- 고용노동부
- 국토교통부
- 환경부
- 식품의약품안전처
- 외교부
- 국방부
- 산업통상자원부

## 1차 확인 항목
- 공개 페이지 제목
- 공개 페이지 URL
- 공개 방식(공지사항/입찰공고/위탁기관 지정공고/공모/기타)
- 공개일
- 민간위탁사무명
- 수탁기관명
- 위탁기간
- 법적 근거
- 비고(비공개 사유/예외 여부)

## 수집 원칙
- 홈페이지 원문 우선
- 검색 결과만으로 단정하지 않음
- 공개 여부가 불명확하면 "추가 확인 필요"로 표기
- 국가안전보장·외교·국방 예외 조항은 별도 표시
